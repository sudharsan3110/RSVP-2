# Creating an Event

Welcome to our RSVP Event SaaS platform! This guide will walk you through the process of creating your first event.

## Step 1: Sign Up / Sign In

1. Visit **[rsvp.kim](http://rsvp.kim/)** and sign in with your email.  
2. If you donâ€™t have an account, create one by signing up with your email.  

## Step 2: Navigate to Event Creation

1. After logging in, go to the **Events** page.  
2. Click **Create Event** (top-right corner).  

## Step 3: Complete Your Profile (New Users)

If your profile is incomplete, update it on the **Profile** page before creating an event.  

## Step 4: Enter Event Details

Fill in the essential details:  
- **Event Title**: Provide a clear and descriptive name.  
- **Category**: Select from options like Hangout, Birthday, Meetup, etc.  
- **Date & Time**: Set the start and end date/time.  
- **Location**:  
  - **In-person**: Enter the venue address.  
  - **Virtual**: Add a meeting link (e.g., Zoom, Google Meet).  
  - **Decide Later**: If the event location is not decided yet.  

## Step 5: Add a Description

Write an engaging event description. Include:  
- Agenda  
- Guest speakers  
- Key details or prerequisites  

## Step 6: Configure RSVP Settings

- Set the maximum number of attendees.  
- Enable/disable approval for attendee requests.  

## Step 7: Set Event Accessibility

- Choose whether the event is **Public** or **Private**.  
  - Public events are visible to all users.  
  - Private events are accessible only via a shared link.  

## Step 8: Upload Media

Add an event image or logo to make your event page visually appealing.  

## Step 9: Review & Publish

1. Review all event details carefully.  
2. Click **Preview** to see how it looks:  
   - On mobile, the preview appears at the bottom of the page.  
   - On desktop, the preview appears on the right side.  
3. When ready, click **Create Event** to publish.

1. Once published, youâ€™ll be redirected to your event page.  
2. Copy the event link from the address bar and share it via email, social media, or other channels.
